# Web3 Startup Idea Generator

AI agent for generating structured Web3 startup ideas using blockchain and decentralized technologies.

## Features
- Generates structured Web3 startup ideas
- Provides tokenomics and revenue models
- Incorporates latest Web3 trends

## How to use
1. Enter a prompt like:  
   `Generate a Web3 startup idea for DePIN in Southeast Asia with token utility and revenue model.`
2. Receive a detailed startup concept with token utility and business plan.
