# Startup Validation Framework

1. Problem
- What real problem exists?
- Who suffers from it?

2. Target Users
- Crypto natives?
- Web2 users entering Web3?
- Developers?

3. Solution
- How blockchain helps?
- Why Web3 is necessary?

4. Product
- MVP features
- Long-term features

5. Token Utility
- Why token is needed
- How it is used

6. Revenue Model
- Fees
- Subscription
- Token-based economy

7. Risks
- Regulation
- Adoption
- Tech risk
